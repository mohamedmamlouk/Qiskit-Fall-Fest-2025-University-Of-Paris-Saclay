## Image Guidelines

- **Format:** JPG for photos, PNG for logos with transparency
- **Quality:** High resolution, but optimized for web
- **Alt text:** Always provide descriptive alt text
- **Accessibility:** Ensure good contrast and readability

## Free Image Sources

- [Unsplash](https://unsplash.com/) - Free high-quality photos

## Optimization

Before uploading, consider optimizing images:
- Use tools like TinyPNG, ImageOptim, or online compressors
- Target file sizes under 500KB for large images
- Use appropriate dimensions (don't upload 4K images for small displays)
