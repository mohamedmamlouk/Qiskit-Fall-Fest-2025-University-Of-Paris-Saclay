⏳ À Faire Ensuite

EN ATTENTE Actions requises pour compléter le site :

1. Configuration GitHub

  • Créer un repository GitHub
  • Uploader tous les fichiers
  • Activer GitHub Pages dans Settings
  • Tester l'URL GitHub Pages

2. Contenu Personnalisé

  • Créer un Google Form pour l'inscription
  • Remplacer l'URL du formulaire dans index.html
  • Ajouter les vraies photos des speakers
  • Ajouter les vraies photos des organisateurs
  • Télécharger les logos IBM Quantum et Qiskit
  • Mettre à jour les dates et informations de l'événement

3. Tests et Optimisation

  • Tester sur mobile et desktop
  • Vérifier tous les liens
  • Optimiser les images
  • Configurer Google Analytics (optionnel)

🚀 Serveur Local

PRÊT Pour tester localement, utilisez une de ces commandes :

# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000

Puis ouvrez : http://localhost:8000
